What is Augmented Reality?

Augmented reality, unlike virtual reality, is taking the world that we already live in a make it better. Augmented reality essentially overlays information on the real world.

To overlay the information, a camera will identify the distance and the angle an object is from the camera. The camera is able to do this through the use of different sensors. Then the projects digital information on the image which allows the user to augmented reality.
